Introdução

Olá, pessoal! Neste blog, vou compartilhar com vocês um pouco sobre HTML e suas tags. Como alguém que já tem algum tempo de experiência na web, posso dizer que o HTML é a espinha dorsal de qualquer página da web. Vamos começar!

Tags HTML

O HTML (Hypertext Markup Language) é uma linguagem de marcação usada para criar a estrutura e o conteúdo de uma página da web. As tags HTML são elementos que envolvem o conteúdo e fornecem informações sobre como esse conteúdo deve ser exibido no navegador.

  • <h1> até <h6>: São usadas para criar títulos e subtítulos em ordem de importância.
  • <p>: Cria parágrafos de texto.
  • <mark>: Realça o texto com um fundo amarelo.
  • <small>: Define o texto como menor.
  • <i>: Coloca o texto em itálico.
  • <u>: Sublinha o texto.
  • <strong>: Torna o texto mais enfatizado.
  • <ol>, <ul>, <li>: Criam listas ordenadas e não ordenadas com itens.
  • <a>: Cria links para outras páginas.
  • <hr>: Insere uma linha horizontal para separar conteúdo.
  • <sub>: Exibe o texto como subscrito.
  • <sup>: Exibe o texto como sobrescrito.
  • <blockquote>: Cria uma citação em bloco.
  • <font>: Usada para definir o tamanho, a cor e o tipo de fonte (embora não seja mais recomendada).
  • <del>: Indica texto que foi removido ou deletado.
  • <abbr>: Define uma abreviação que pode ser explicada quando o mouse passa sobre ela.

Estrutura Básica

Uma página HTML geralmente começa com a declaração do tipo de documento (DOCTYPE) seguida da tag '<html>', que contém as tags '<head>' e '<body>'. O '<head>' contém informações sobre a página, como o título e os metadados, enquanto o '<body>' contém o conteúdo visível da página.

Exemplo

Aqui está um exemplo simples de um trecho de código HTML:

                
                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>Minha Página</title>
                    </head>
                    <body>
                        <h1>Bem-vindo ao meu site</h1>
                        <p>Esta é uma página de exemplo</p>
                    </body>
                    </html>
